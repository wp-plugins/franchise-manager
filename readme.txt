This Plugin will help you to easily create a Franchise manager into your WordPress website.

== Description ==

This Plugin will help you to easily create a franchise manager system into your WordPress website. The Franchise System will display a List of franchise and details of franchise users to see availability franchise in wordpress.

= Features =
If you want new features or functionalities in the Franchise System drop us a line in [support forum](http://wordpress.org/support/plugin/franchise-manager).

**Responsive**

* Front End Franchise manager is responsive and can be viewed on all browsers and devices.

== Installation ==

Upload the folder **franchise-manager** from the zip file to "wp-content/plugins/" and activate the plugin in your admin panel or upload **franchise-manager.zip** in the "Add new" section.

Shortcode in posts:
`[franchise_list_basic]`
